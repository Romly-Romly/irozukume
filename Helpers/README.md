# Helpers

特定のレイヤーに属さず、複数の場所から共通して使う「状態を持たない補助」と「軽量なインフラ」を置くフォルダ。`Models`(ドメインのデータ型)・`Services`(状態や入出力を伴うサービス)・`Controls`/`Views`/`ViewModels`(UI)・`Interop`(ネイティブ相互運用)のいずれにも属さない横断的な道具がここに入る。

## 収録の基準

- 状態を持たない、もしくは生成されたら不変な小さな型であること。
- 入出力や OS への依存を持たず、純粋な計算・変換・組み立てに徹すること。
- 特定の画面やモデルに紐付かず、複数の場所から同じ式・同じ振る舞いで使われること。

これらに当てはまらないものは、データなら `Models`、状態や入出力を伴うなら `Services`、ネイティブ呼び出しなら `Interop` へ置く。
