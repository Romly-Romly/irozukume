# イロヅクメ (Irozukume)

> 🌐 **[English README →](README.md)**

タスクトレイ常駐型の色作成／カラーピッカーです。様々なカラーモデルで色を作成したり、簡単な配色を作ったり、デスクトップ上の色を拾うことが出来ます。WCAG ガイドライン準拠の文字のコントラストチェック機能もあります。

昔似たようなフリーウェアがありまして、その現代版といった体を目指しております。

> **これはめっちゃアルファ版です。** かろうじて動く程度のものとお考え下さい。



## スクリーンショット

![RGB・CMYK タブ](screenshots/rgb.png)

*RGB, CMYK, HSV, HSL, YUVなど基本的な表色系をサポート*

![OKLCH タブ](screenshots/oklch.png)

*モダンな表色系として、LCH, LAB, HWB なども搭載*

![ミックス タブ](screenshots/mix.png)

*作った色を好きな位置に配置し、グラデーションから色を選ぶこともできます*

![配色 タブ](screenshots/scheme.png)

*補色、スプリットコンプリメンタリー、トライアードなどの代表的な配色機能も*



## ダウンロード

[最新リリース](https://github.com/Romly-Romly/irozukume/releases/latest) から zip をダウンロードして下さい。

- **ポータブル版** — `Irozukume-x64-v*.zip`（64bit / x64）

展開して `Irozukume.exe` を実行するだけで動きます。.NET ランタイム等の事前インストールは不要です（アプリに同梱しています）。

本アプリはコード署名をしていないため、初回起動時に Windows SmartScreen の警告が表示されます。「詳細情報」をクリックし、「実行」を選ぶと起動できます。自己責任でどうぞ。

## 設定

### 設定の保存先

設定内容は **実行ファイルと同じフォルダ** に `settings.json` として保存されます。設定を消したい場合は、この `settings.json` を削除して下さい。



## 動作環境

| OS | バージョン |
|---|---|
| Windows | 10 バージョン 2004 (build 19041) 以降 / 11、64bit (x64) |

画面キャプチャ系 API の都合で Windows 10 バージョン 2004 以降が必要です。アルファ版は x64 のみの提供です。
また一部のアプリからのカラーピッカーを有効にするには管理者として起動する必要があります。
